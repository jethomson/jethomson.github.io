0001 %FLAT_CORRECT - Corrects for variation in pixel sensitivity and vignetting.
0002 %
0003 % A flat frame is a record of the variation in sensitivity from pixel to pixel
0004 % and the drop off of intensity at the image's border caused by the lens.
0005 %
0006 % Syntax:  I = flat_correct(light, flat)
0007 %
0008 % Inputs:
0009 %    light - a spectrograph.
0010 %    flat  - the master flat frame.
0011 %
0012 % Outputs:
0013 %    I - a spectrograph that has been corrected for unevenness across it's
0014 %        entire field.
0015 %
0016 % Example:
0017 %    none
0018 %
0019 % Other m-files required: none
0020 % Subfunctions: none
0021 % MAT-files required: none
0022 %
0023 % See also: NONE
0024 %
0025 % Author: Jonathan Thomson
0026 % Work:
0027 % email:
0028 % Website: http://jethomson.wordpress.com
0029 %
0030 
0031 function I = flat_correct(light, flat)
0032 
0033     if (nargin ~= 2)
0034         usage('flat_correct(light, flat)');
0035     end
0036 
0037     I = light./flat;
0038     % the flat will have zeros because of dark subtraction
0039     I(isnan(I)) = 0; % handles 0/0
0040     I(isinf(I)) = 0; % handles +/- n/0
0041 
0042 end
